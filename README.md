# Customer Support Hub

A backend service for managing customer support interactions, built with Spring Boot 3.x, Spring Data JPA, and MySQL.

Supports three role types — **ADMIN**, **AGENT**, and **CUSTOMER** — with role-based access to customer profiles and tickets.

## Status

🚧 Work in progress.

## Tech stack

- Java 21
- Spring Boot 3.x (Spring MVC, Spring Data JPA, Spring Validation, Spring Security)
- MySQL
- Docker / Docker Compose
